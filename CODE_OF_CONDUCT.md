# Contributor Covenant 行为准则 / Code of Conduct

## 我们的承诺 / Our Pledge

为了营造开放、友好的环境，我们作为贡献者与维护者承诺：让参与本项目和社区的每一个人都不受骚扰，无论年龄、体型、可见或不可见的残疾、族裔、性别认同与表达、经验水平、教育背景、社会经济地位、国籍、个人外貌、种族、宗教，或性取向。

我们承诺以营造开放、热情、多元、包容、健康的社区为目标进行行动与互动。

## 我们的准则 / Our Standards

促进社区积极氛围的行为示例：

- 对他人展现同理心与善意
- 尊重不同的观点与经验
- 给予并优雅地接受建设性反馈
- 对错误负责，向受影响者致歉，并从中学习
- 关注不仅是个人，也是整个社区的最大利益

不可接受的行为示例：

- 使用带有性暗示的语言或图像，以及任何形式的性骚扰
- 嘲讽、侮辱性或贬低性的评论，人身攻击或政治攻击
- 公开或私下骚扰
- 未经明确许可发布他人的私人信息（如住址、邮箱）
- 在职业场合中，其他可被合理认定为不当的行为

## 执行职责 / Enforcement Responsibilities

社区维护者负责澄清并执行本准则，并对其认为不当、有威胁、冒犯或有害的行为采取适当且公平的纠正措施。

社区维护者有权且有责任删除、编辑或拒绝任何不符合本准则的评论、提交、代码、wiki 编辑、issue 或其他贡献，并在适当时说明原因。

## 适用范围 / Scope

本准则适用于所有社区空间，也适用于个人在公开场合代表社区时。代表社区的情形包括但不限于：使用官方电子邮件地址、通过官方社交媒体账号发帖、作为指定代表参加线上或线下活动。

## 执行 / Enforcement

如遇骚扰、辱骂或其他不可接受的行为，请联系本项目维护者：

- 通过 GitHub 私信 [@hawkli-1994](https://github.com/hawkli-1994)
- 或通过 [Security Advisory](https://github.com/rongxinzy/SwarmMind/security/advisories/new) 渠道（标注「行为准则相关」）

所有投诉都会被及时、公正地审查与调查。所有社区维护者有义务尊重举报者的隐私与安全。

## 执行准则 / Enforcement Guidelines

社区维护者将根据以下指引判断违反本准则行为的后果：

### 1. 纠正 / Correction

**社区影响**：使用了不当语言或被认为不专业、不受欢迎的行为。

**后果**：来自维护者的私下书面警告，说明违规性质并解释为何不当。可能会要求当事人公开道歉。

### 2. 警告 / Warning

**社区影响**：通过单次或一系列行为造成违规。

**后果**：附带后续行为约束的警告。在指定时间内，不得与相关人员（包括主动联系准则执行者）发生互动。违反者可能会被临时或永久封禁。

### 3. 临时封禁 / Temporary Ban

**社区影响**：严重违反社区准则，包括持续的不当行为。

**后果**：在指定时间内，禁止与社区进行任何形式的互动或公开沟通。

### 4. 永久封禁 / Permanent Ban

**社区影响**：表现出违反社区准则的模式，包括持续的不当行为、骚扰个人，或攻击 / 贬低特定群体。

**后果**：永久禁止在社区内进行任何形式的公开互动。
