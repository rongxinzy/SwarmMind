# SwarmMind Documentation

This directory contains the product and architecture documents for SwarmMind as an enterprise agent control plane.

## Structure

- `product-positioning.md`: ICP, problem framing, product boundaries, and differentiation
- `architecture.md`: system architecture, control/data plane design, governance model
- `roadmap.md`: phased roadmap with priorities and delivery outcomes
- `archive/README_v1_vision.md`: archived narrative from earlier "multi-agent + shared memory" framing
