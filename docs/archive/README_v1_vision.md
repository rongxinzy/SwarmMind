## Why It Was Archived

The framing above was directionally strong but too easy to confuse with generic multi-agent collaboration frameworks. The project narrative has shifted to emphasize:

- enterprise control plane positioning,
- governed context instead of raw shared memory,
- execution trace and auditability over free-form reasoning exposure,
- adaptive policy learning with enterprise-safe rollout.

## Migration Mapping

- Shared Context → Governed Context Layer
- Self-Evolution → Adaptive Routing & Policy Learning
- Reasoning Trace → Execution / Collaboration Trace
- LLM-rendered status → Context-to-View Compiler
