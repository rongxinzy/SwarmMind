# 资源库、知识库、定时任务与设置

> 日期：2026-03-27
> 用途：定义除项目主线外的资产和系统管理页面

## 1. 页面一：资源库

### 1.1 页面卡片

- 页面名：`资源库`
- 目标用户：所有有资产访问权限的成员
- 页面目标：集中浏览跨项目产物和导出结果
- 页面入口：侧边栏 `资源库`
- 核心动作：
  - 搜索 artifact
  - 按项目 / 类型 / 作者筛选
  - 打开来源项目

### 1.2 信息结构

- 顶部：搜索和筛选
- 中部：资源表格 / 卡片
- 右侧：预览和元数据

### 1.3 桌面端线框

```text
+------------------------------------------------------------------------------------------------+
| 资源库                                                     [搜索资源] [筛选] [导出索引]       |
+--------------------------------+----------------------------------------------------------------+
| 资源列表                       | 预览与元数据                                                   |
| CRM / PRD v0.3                 | 标题：PRD v0.3                                                 |
| CRM / 风险清单                 | 项目：新一代 CRM 平台建设                                      |
| 招聘 / Offer 模板              | 类型：文档                                                     |
| 销售培训 / 课程大纲            | 作者：产品经理 Agent                                           |
|                                | [打开项目] [查看审计]                                          |
+--------------------------------+----------------------------------------------------------------+
```

### 1.4 状态设计

- 空状态：提示资源会随着项目推进自动累积
- 加载状态：列表和预览骨架
- 错误状态：预览失败时保留元数据
- 权限不足状态：只展示可访问条目

## 2. 页面二：知识库

### 2.1 页面卡片

- 页面名：`知识库`
- 目标用户：全体成员；部分知识需要权限
- 页面目标：浏览企业知识资产并作为项目上下文来源
- 页面入口：侧边栏 `知识库`
- 核心动作：
  - 搜索知识条目
  - 查看分类
  - 引用到项目或对话

### 2.2 信息结构

- 左侧：分类树
- 中间：搜索结果 / 文档列表
- 右侧：预览与引用动作

### 2.3 桌面端线框

```text
+------------------------------------------------------------------------------------------------+
| 知识库                                                     [搜索知识] [新建引用]              |
+------------------------------+--------------------------------+--------------------------------+
| 分类树                       | 结果列表                       | 文档预览与引用                  |
| - 行业资料                    | 制造业客户画像指南             | 可见级别：管理层批准后开放      |
| - 技术规范                    | 企业 SSO 接入标准              | 最近被引用：CRM 平台建设        |
| - 合规政策                    | 数据分级要求                   | [引用到项目] [申请访问权限]     |
+------------------------------+--------------------------------+--------------------------------+
```

### 2.4 关键交互

1. 从知识库引用到项目时，系统要求选择目标项目。
2. 若当前用户无权访问，显示 `申请访问权限`，而不是隐藏条目来源。

### 2.5 状态设计

- 空状态：暂无知识条目或当前筛选无结果
- 加载状态：结果列表骨架
- 错误状态：搜索失败可重试
- 权限不足状态：内容模糊化显示，并给出申请动作

## 3. 页面三：定时任务

### 3.1 页面卡片

- 页面名：`定时任务`
- 目标用户：管理员、项目负责人、高级用户
- 页面目标：配置周期运行、自动报告和例行检查
- 页面入口：侧边栏 `定时任务`
- 核心动作：
  - 新建定时任务
  - 编辑执行周期
  - 查看最近执行记录

### 3.2 信息结构

- 顶部：新建按钮和筛选
- 中部：任务表格
- 右侧：任务详情与最近运行

### 3.3 桌面端线框

```text
+------------------------------------------------------------------------------------------------+
| 定时任务                                                     [全部] [启用中] [+ 新建任务]      |
+------------------------------------------------------------------------------------------------+
| 名称                    | 目标对象             | 周期         | 最近执行      | 状态   | 操作   |
| 每周项目健康摘要         | 工作台组织视图       | 周一 09:00   | 今天 09:00    | 启用   | 查看   |
| CRM 风险巡检            | CRM 平台建设         | 每天 18:00   | 今天 18:00    | 启用   | 查看   |
| 招聘周报                | 招聘流程优化         | 周五 17:00   | 上周五        | 暂停   | 查看   |
+------------------------------------------------------------------------------------------------+
| 右侧详情：输入模板 / 输出目标 / 最近运行日志 / 失败原因                                  |
+------------------------------------------------------------------------------------------------+
```

### 3.4 关键交互

1. 新建定时任务时，需要选择目标对象是 `项目`、`工作台聚合视图` 还是 `资源库导出`。
2. 最近执行失败时，右侧显示失败摘要和重试入口。

### 3.5 状态设计

- 空状态：引导用户创建第一个周期任务
- 加载状态：表格骨架
- 错误状态：执行记录加载失败不影响任务表
- 权限不足状态：无编辑权限时只读

## 4. 页面四：设置

### 4.1 页面卡片

- 页面名：`设置`
- 目标用户：所有登录用户；部分模块仅管理员可编辑
- 页面目标：管理账户、组织、通知和偏好设置
- 页面入口：底部账户区
- 核心动作：
  - 查看个人信息
  - 调整通知
  - 切换组织
  - 查看权限说明

### 4.2 信息结构

- 左侧：设置子导航
- 中间：设置表单
- 右侧：帮助说明

### 4.3 桌面端线框

```text
+------------------------------------------------------------------------------------------------+
| 设置                                                                               [保存]      |
+------------------------------+--------------------------------+--------------------------------+
| 子导航                       | 设置内容                       | 帮助说明                        |
| - 账户                        | 姓名 / 邮箱 / 头像             | 你的可见权限来自组织角色        |
| - 通知                        | 审批通知 [x]                  |                                  |
| - 组织与权限                  | 项目阻塞通知 [x]              | 当前组织：Default Org           |
| - 外观                        | 主题：跟随系统                 |                                  |
+------------------------------+--------------------------------+--------------------------------+
```

### 4.4 状态设计

- 空状态：无
- 加载状态：表单骨架
- 错误状态：字段级错误
- 权限不足状态：组织级设置只读

## 5. 设计结论

这些页面虽然不直接承担主工作流，但必须服务于主闭环：

- 资源库承接 artifact 聚合
- 知识库承接上下文来源与权限申请
- 定时任务承接自动化执行
- 设置承接账户与组织治理
